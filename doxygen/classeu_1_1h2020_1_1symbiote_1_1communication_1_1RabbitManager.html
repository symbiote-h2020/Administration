Detailed Description

Class used for all internal communication using RabbitMQ AMQP implementation. It works as a Spring Bean, and should be used via autowiring.

RabbitManager uses properties taken from CoreConfigServer to set up communication (exchange parameters, routing keys etc.)

void eu.h2020.symbiote.communication.RabbitManager.sendRpcMessage ( String  exchangeName,
String  routingKey,
String  message,
IRpcResponseListener  responseListener

Method used to send message via RPC (Remote Procedure Call) pattern. Before sending a message, a temporary response queue is declared and its name is passed along with the message. When a consumer handles the message, it returns the result via the response queue. When the response is available, the responseListener is notified of its arrival.
